# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts

In the project directory, you can run:

### `npm i` and then `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will open the landing page which displays the list of categories.


### `http://localhost:3000/category/<id>`

Displays the products under a particular category. On adding an item to the cart, if the user is not signed in it will redirect to the login page.

### `http://localhost:3000/sign-in`

Displays the sign in page.

### `http://localhost:3000/sign-up`

Displays the sign up page.If the user does not have an account then he/she can sign up for one.

### `http://localhost:3000/cart`

Displays the list of products in the cart.

### `http://localhost:3000/checkout`

Displays the checkout page where users can enter their checkout information.

### `http://localhost:3000/order-confirmation`

Displays the order confirmation once the order is placed successfully.

### `http://localhost:3000/orders`

Displays the list of orders placed.
